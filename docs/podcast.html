How Things Work

Exploring how blockchains (and the mechanisms that power them) work.


How Things Work #5: TEEs All The Way Down

On this episode of How Things Work, Jack Kearney, Arnaud Brousseau, and Zeke Mostov from Turnkey discuss how to architect secure applications with TEEs (and yes, there are insecure applications using TEEs).

We cover when and where TEEs are useful alongside MPC and ZK, the foundations needed before deploying them, and how to safely handle code updates and external state.

Guests: Jack Kearney, Arnaud Brousseau, Zeke Mostov - Turnkey
Find on: YouTube, Spotify, Apple Podcasts
How Things Work #4: The Quick Merkle Database and Fast Ahead-of-Formation Optimization

On this episode of How Things Work, Ryan Zarick, Isaac Zhang, and Thomas Kim from LayerZero discuss scaling blockchain throughput with their Quick Merkle Database (QMDB) and Fast Ahead-of-Formation Optimization (FAFO).

We explore how QMDB works, why its proof structure is well suited for ZK applications, and how FAFO builds on QMDB to push toward the 1M TPS milestone on a single machine.

Guests: Ryan Zarick, Isaac Zhang, Thomas Kim - LayerZero
Find on: YouTube, Spotify, Apple Podcasts
How Things Work #3: Scaling Broadcast with Raptor Codes

On this episode of How Things Work, Kushal Babel and Babak Gilkalaye from Category Labs discuss their work on scaling blockchain broadcast.

We look at how today's networks like Bitcoin, Ethereum, and Solana handle data dissemination, then dive into Monad's "broadcast stack" and its use of Raptor Codes to balance latency and throughput, before examining the tradeoffs of deploying pre-consensus broadcast.

Guests: Kushal Babel, Babak Gilkalaye - Category Labs
Find on: YouTube, Spotify, Apple Podcasts
How Things Work #2: Executable Semantic Frameworks and K

On this episode of How Things Work, Grigore Rosu reflects on two decades of runtime verification and introduces Pi Squared, a project using Executible Semantic Frameworks to build blockchain execution environments that can support any programming language (even ones you write yourself).

Guests: Grigore Rosu - Pi Squared
Find on: YouTube, Spotify, Apple Podcasts
How Things Work #1: Partially Ordered Datasets

On this episode of How Things Work, Shresth Agrawal discusses pod's research on Partially Ordered Datasets and their use in building low-latency blockchain applications.

We cover pod's origins, how applications can function without total ordering, and how their recent work enables censorship-resistance and accountability without needing to run traditional consensus mechanisms.

Guests: Shresth Agrawal - pod
Find on: YouTube, Spotify, Apple Podcasts
How Things Work #0: ZODA and The Accidental Computer

The pilot episode of How Things Work features Alex Evans and Guillermo Angeris from Bain Capital Crypto unpacking blockchain scaling research.

We discuss Data Availability Sampling (DAS) and their ZODA framework before discussing the "Accidental Computer", a surprising result on polynomial commitment schemes and what it means for future blockchain design.

Guests: Alex Evans, Guillermo Angeris - Bain Capital Crypto
Find on: YouTube, Spotify, Apple Podcasts
